# Demo Script

## Story-Driven Golden Flow
1. Start with a normal Eid-eve outlet state.
2. Show deceptive total: aggregate looks healthy while one provider runway is short.
3. Show provider-specific runway clock and shared-cash runway.
4. Show evidence fingerprint for unusual activity.
5. Show degraded confidence for stale/missing/conflicting data.
6. Show normal Eid/salary-day spike that does not become a review case.
7. Route alert to owner.
8. Acknowledge, assign/escalate, review, add notes, resolve, close.
9. Show metrics.
10. Reset and replay the scenario.

## Safe Demo Language
Use: "The pattern is consistent with expected demand; no additional review is currently recommended."

Use: "Coordinate approved liquidity support through the responsible provider operations channel."
