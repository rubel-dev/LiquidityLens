# Demo Script

## Story-Driven Golden Flow
1. Start with a normal Eid-eve outlet state. `SCN-001`
2. Show deceptive total: aggregate looks healthy while one provider runway is short. `SCN-001`
3. Show provider-specific runway clock and shared-cash runway. `SCN-001`
4. Show evidence fingerprint for unusual activity. `SCN-002`
5. Show degraded confidence for stale/missing/conflicting data. `SCN-003`
6. Show normal Eid/salary-day spike that does not become a review case. `SCN-005`
7. Route alert to owner. `SCN-004`
8. Acknowledge, assign/escalate, review, add notes, resolve, close. `SCN-004`
9. Show metrics. `SCN-004`
10. Reset and replay the scenario. `SCN-001`, `SCN-002`, `SCN-004`
